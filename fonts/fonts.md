# Modulbit typography

## Primary font 
Main font used across the website and UI.

- Font: Inter
- Weights: 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)

## Secondary font for headings and branding

- Font: Poppins
- Weights: 500, 600

## Logo Font
Used for logo and brand mark.

- Font: Montserrat
- Weight: SemiBold
- Notes: Customized for logo (modified shapes and spacing)

---

## Usage Guidelines

- Use Inter for most text (clean and readable)
- Use Poppins or Montserrat for headings if needed
- Keep typography simple and consistent
- Avoid using too many different fonts

---

## Text Styles

### Headings
- H1: 32–40px, Bold
- H2: 24–32px, SemiBold
- H3: 18–24px, Medium

### Body Text
- Size: 14–16px
- Weight: Regular (400)

### Small Text
- Size: 12–13px
- Color: Gray

---

## Web Usage Example

```css
body {
  font-family: 'Inter', sans-serif;
}

h1, h2, h3 {
  font-family: 'Poppins', sans-serif;
}
